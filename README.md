# Boda Idoia & Joan

Single-page wedding website built with React + Vite.

## Local development

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Production preview

```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

Then set GitHub → Settings → Pages:
- Source: Deploy from a branch
- Branch: `gh-pages` (root)
